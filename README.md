Overview:
This repository contains my learning and implementation of Image Processing and Computer Vision using OpenCV, where I focused not just on using functions, but understanding how things actually work internally. The work includes core techniques like edge detection, line detection, filtering, and feature-based image processing.

Key Concepts Implemented:
Canny Edge Detection
Hough Transform (Line Detection)
HSV Color Space Filtering
Gaussian (Low-Pass) Filtering
High-Pass Filtering (Sharpening)
Object Tracking & Face Detection
RANSAC Algorithm (Image Stitching)

What I Explored:
How edges are detected using intensity gradients (Ix, Iy)
How Hough Transform maps lines and detects intersections
How HSV thresholding isolates regions of interest
How filters affect image smoothing and sharpening
How RANSAC identifies matching features and stitches images

Applications:
Lane detection (used in AGV project)
Image masking and manipulation
Panorama creation using RANSAC
Real-time vision-based systems

Tech Stack:
Python
OpenCV
NumPy

Learning Outcome:
This repository reflects my transition from understanding what happens (graphics tools) to how it works (algorithms) and finally applying it in real systems (robotics projects).
